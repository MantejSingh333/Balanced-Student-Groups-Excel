# Balanced Student Groups (Excel)

Excel-based project creating balanced student groups based on performance and communication metrics to improve collaboration dynamics.

## Overview
Aggregated individual assignment scores (M1, M2, M3) into a total performance metric and combined it with qualitative communication data (“Often” vs. “Not Often”). Distributed students across groups to ensure:

- Even performance levels  
- Equal spread of highly communicative students  
- No group disproportionately strong or weak  

Final outputs included group-level performance totals to verify balance and fairness.

## Workflow & Tools
- Used **Excel formulas**, including **VLOOKUP**, sorting, and manual validation  
- Structured qualitative data for analysis  
- Verified group balance with aggregated metrics  

## Skills
Microsoft Excel, VLOOKUP, data aggregation & sorting, problem-solving, translating qualitative data into actionable insights
