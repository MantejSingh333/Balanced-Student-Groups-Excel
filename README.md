# Balanced-Group-Assignment-Using-Excel-VLOOKUP-Performance-Metrics-

Project Summary
In this task, I used Excel to analyze student performance and communication behavior in order to create balanced groups for collaborative work. I aggregated individual assignment scores (M1, M2, M3) into a total performance metric and combined it with qualitative communication data (“Often” vs. “Not Often”).

Using Excel formulas, including VLOOKUP, sorting, and manual validation, I distributed students across multiple groups to ensure:

Even performance levels across all groups

An equal spread of highly communicative students

No group was disproportionately strong or weak

This approach improved collaboration dynamics by preventing clustering of top performers or frequent communicators in a single group. The final output included group-level performance totals to verify balance and fairness.
